Politica de Privacidade

Aperfeicoar Mensagens IA - atualizada em 30 de abril de 2026.

Esta extensao ajuda o usuario a melhorar textos em campos de mensagem. Ela nao e afiliada, patrocinada ou endossada por WhatsApp, Gmail, Instagram, Google ou Meta.

Quais dados sao coletados

  • Texto enviado pelo usuario quando ele clica em Aperfeicoar, Gerar reescrita ou Gerar variacoes.
  • Texto de mensagens recebidas visiveis quando o usuario ativa a traducao automatica de recebidas.
  • Configuracoes escolhidas pelo usuario, como idioma, modo de uso, perfil de reescrita e preferencias de aprendizado.
  • Dados de conta, quando houver login, como email, identificador de usuario, plano e status de uso.
  • Metadados tecnicos necessarios, como dominio do site, origem chrome_extension, tipo de mensagem classificado e contadores de uso.
  • Eventos de aprendizado, se o usuario permitir, como sugestao aceita, rejeitada, variacao escolhida e feedback positivo ou negativo.

Como os dados sao usados

  • Gerar a versao melhorada do texto solicitado pelo usuario.
  • Traduzir mensagens recebidas para o idioma escolhido quando o recurso estiver ativado.
  • Aplicar limites de uso por plano Free, Pessoal, Empresarial ou Premium.
  • Manter preferencias e melhorar a qualidade das sugestoes, quando o aprendizado estiver ativado.
  • Permitir login, checkout, suporte, seguranca, prevencao de abuso e funcionamento do servico.

Texto e dados sensiveis

O texto digitado pode conter dados pessoais ou sensiveis, dependendo do que o usuario escrever. A extensao nao envia texto automaticamente para reescrita. No recurso de traducao de recebidas, mensagens visiveis da conversa so sao enviadas ao backend quando o usuario ativa explicitamente essa opcao.

Por padrao, textos nao sao salvos no backend para aprendizado. Se o usuario ativar a opcao "Permitir salvar textos para melhorar analise", textos originais e gerados poderao ser armazenados para melhorar o produto. O usuario pode desativar essa opcao a qualquer momento.

Armazenamento local

A extensao usa chrome.storage para salvar configuracoes, sessao, historico recente local das ultimas mensagens melhoradas e feedback. Esses dados ficam no navegador do usuario, exceto quando uma acao da extensao envia informacoes ao backend para executar a funcionalidade solicitada.

Compartilhamento

Os dados podem ser processados por provedores necessarios ao funcionamento do produto, como infraestrutura de backend, provedor de IA, autenticacao e pagamento. Dados de pagamento sao processados pelo provedor de pagamento; a extensao nao coleta numero de cartao.

Os dados nao sao vendidos, nao sao usados para publicidade comportamental e nao sao transferidos para finalidades nao relacionadas ao funcionamento ou melhoria do produto.

Limited Use

O uso de dados obtidos pela extensao e limitado a fornecer ou melhorar recursos visiveis ao usuario, como melhoria de texto, controle de uso, preferencias, aprendizado permitido pelo usuario, seguranca e suporte. A extensao nao usa dados para anuncios, revenda de dados ou avaliacao de credito.

Controle do usuario

  • O usuario pode usar o modo Free sem login, sujeito ao limite diario.
  • O usuario pode desativar aprendizado personalizado no painel.
  • O usuario pode desativar armazenamento de texto para aprendizado no painel.
  • O usuario pode sair da conta e remover dados locais desinstalando a extensao ou limpando os dados do navegador.

Seguranca

A versao publicada deve usar somente backend HTTPS. Tokens de sessao ficam armazenados localmente no navegador e sao enviados apenas para o backend configurado para autenticar chamadas da API.

Contato

Para publicar na Chrome Web Store, substitua este trecho pelo email oficial de suporte e pela URL publica desta politica de privacidade.
